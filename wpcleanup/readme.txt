Clean up default WordPress content and optionally reset your site to its default state.

== Description ==

WP Cleanup is a powerful tool that helps you:

* Remove default WordPress content (Hello World post, sample page, first comment)
* Clean up all posts and pages
* Remove unnecessary plugins
* Reset WordPress to its default state

== Installation ==

1. Upload the `wpcleanup` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tools > WP Cleanup to access the cleanup options

== Usage ==

The plugin provides four main functions:

1. Clean Default Content: Removes the default WordPress content
2. Clean All Posts: Removes all posts, pages, and comments
3. Clean All Plugins: Removes all plugins except WP Cleanup
4. Reset to Default State: Completely resets WordPress while maintaining admin access

== Frequently Asked Questions ==

= Is this action reversible? =

No, all cleanup actions are permanent. Please make sure to backup your database before using this plugin.

= Will this affect my admin account? =

No, your admin account will be preserved even when using the complete reset function.
